# Systek V2.5

TUI d'administration Linux simple, professionnelle et orientée admin système.

## Installation

```bash
sudo ./install.sh
```

## Lancement

```bash
systek
sudo systek
```

Sans sudo, certaines fonctionnalités restent en lecture seule.

## Commandes utiles

```bash
systek --version
systek --doctor
sudo systek --update
sudo systek --uninstall
```
